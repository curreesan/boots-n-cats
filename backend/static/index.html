Boots and Cats API

v0.1.0
Interactive docs at /docs
Auth
POST/auth/registerCreate an account, logs in immediately
POST/auth/loginLog in with email + password
POST/auth/logoutClear auth cookies
GET/auth/meCurrent logged-in user
POST/auth/refreshMint a new access token
Products
GET/productsList products, filter by species/category
GET/products/:idSingle product
GET/products/categories/allDistinct categories
Pets
GET/petsList pets, filter by species
GET/pets/:idSingle pet profile
GET/pets/breeds/allDistinct breeds
Cart
GET/cartYour cart — auth required
POST/cartAdd an item (upserts quantity) — auth required
PATCH/cart/:product_idSet exact quantity — auth required
DELETE/cart/:product_idRemove an item — auth required
DELETE/cartClear the whole cart — auth required
Chat
POST/chatPlain request/response agent turn — auth required, no picker/confirm-button support (see /ws/chat)
WS/ws/chatAgentic chat — search catalog/knowledge, add to cart, book a consultation, place an order — auth via cookie at connect time
Orders
POST/ordersCheckout — auth required
GET/ordersYour order history — auth required
GET/orders/:idSingle order + items — auth required
Adoption
POST/adoption-consultationsRequest a consultation — auth required
GET/adoption-consultationsYour own requests — auth required
Admin — staff only
POST/admin/productsCreate a product
PATCH/admin/products/:idUpdate a product
DELETE/admin/products/:idDelete a product
POST/admin/petsCreate a pet listing
PATCH/admin/pets/:idUpdate a pet listing
DELETE/admin/pets/:idDelete a pet listing
GET/admin/adoption-consultationsAll requests, oldest first
POST/admin/knowledge/uploadUpload a knowledge doc — chunks, embeds, upserts to Pinecone
GET/admin/knowledgeList knowledge docs + ingestion status
